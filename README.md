# 🌡️ Delhi HeatShield

Delhi HeatShield is a real-time heat vulnerability monitoring platform that maps 
risk across Delhi's 280+ wards using climate data, satellite imagery, and 
demographic indicators — identifying who is most at risk before extreme heat 
turns fatal.

## Features
- 🗺️ Interactive Heat Vulnerability Index (HVI) risk map
- 📊 Ward-level vulnerability scoring across 11 districts
- 🔔 Multi-channel alert system (WhatsApp, SMS, Municipal triggers)
- 👤 User authentication with secure HttpOnly cookie sessions
- 📬 Contact system for emergency reporting and partnerships
- 📱 Fully responsive across mobile and desktop

## Tech Stack
**Frontend:** React, TypeScript, Tailwind CSS, Framer Motion, Vite  
**Backend:** Python, Flask, Flask-JWT-Extended, bcrypt  
**Database:** Supabase (PostgreSQL)  
**Auth:** JWT via HttpOnly cookies

## Purpose
Built to support Delhi's most heat-exposed communities — outdoor workers, 
elderly residents, and informal settlement dwellers — by giving city officials 
and citizens real-time visibility into heat risk zones.
